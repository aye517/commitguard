# CommitGuard / EasyTest Vision

**한 줄**: 테스트를 쉽게 시작하게 해주는 개발자 도구

---

## 핵심 철학

대부분의 테스트 도구는 **테스트 잘하는 사람**을 위한 도구다.

이 프로젝트는 **테스트 안 하는 사람**을 위한 도구다.

---

## 타깃 사용자

- 테스트를 한 번도 써본 적 없는 개발자
- 무엇을 테스트해야 하는지 모르는 개발자
- 테스트 파일 만들기 귀찮은 개발자
- 커버리지 신경 안 쓰는 개발자

→ 그래서 테스트 자체를 안 쓰게 되는 사람

---

## 우리가 주는 경험

```
Write code normally
    ↓
npx easytest
    ↓
tests appear
```

테스트를 **쉽게 시작**하게 해준다.
